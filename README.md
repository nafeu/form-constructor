# Form Constructor

Build HTML forms with customizable schemas

![Form Constructor By Nafeu](https://nafeu.com/form-constructor/app-preview.png)

## Installation

```
git clone https://github.com/nafeu/form-constructor.git
cd form-constructor
npm install
```

## Development

```
npm start
```

## Deployment

```
npm run deploy
```
